# 実行方法
pythonの仮想環境を用意したのち,
```
cd star_plot
pip install -r requirements.txt
```
として必要なモジュールをインストールする。

入力したい写真を./実行用/input/image以下に入れて
```
cd 実行用
```
で./実行用に移動し、
```
python3 run.py <写真の名前>
```
とすると、実行できる。

実行すると標準出力にロスの大きさ等の値が表示され、別ウィンドウで、星座の星の形を結んだもの、写真と重ねたものが表示される。

終了するにはCtrl+Cを押す。この時に出てきた写真のウィンドウを消えます。
